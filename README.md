# Royal Cake
Toko kue premium dengan resep kerajaan Inggris, dipadukan sentuhan coklat dan kopi terbaik.

Coba langsung di [sini](https://tikusrich.github.io/royalcake)

Clone repository ini:
```
git clone https://github.com/tikusrich/royalcake.git
```

Clone versi terbaru Bootstrap:
```
git clone https://github.com/twbs/bootstrap ./royalcake/bootstrap
```

Masuk ke dalam proyek:
```
cd royalcake
```

Jalankan SASS untuk memulai development:
```
sass --watch ./scss/main.scss ./style.css
```

Jalankan Live Server untuk preview real-time:
```
live-server
```
